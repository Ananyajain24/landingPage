Step 1: Prerequisites

Hardware Requirements

  • RAM - 4GB
  • CPU - Intel 2.80 GHz, 2P
  • Space - 8-10 GB

Before proceeding, ensure your system meets the following requirements:

  • Install Python (3.8 or above):
    1. Open your browser and go to the Python downloads page.
    2. Click on the Download Python 3.x.x button.
    3. Run the downloaded installer:
      • Windows: Double-click the .exe file, check the box Add Python to PATH, and click Install Now.
      • Linux/macOS: Use your package manager: 
        sudo apt update
        sudo apt install python3 python3-pip
    4. Verify installation: 
      python3 --version
  • Install Node.js (16 or above):
    1. Visit the Node.js official site.
    2. Click LTS (Recommended for most users).
    3. Run the installer and follow the setup wizard.
    4. Verify installation: 
      node --version
       npm --version
  • Install NGINX:
    • Linux: Open your terminal and run: 
      sudo apt update
      sudo apt install nginx
    • macOS: Use Homebrew: 
      brew install nginx
    • Windows: Follow the NGINX installation guide.
    • Start NGINX: 
      sudo service nginx start
  • Install Database Server:
    • Use PostgreSQL as your database server. Follow the PostgreSQL installation guide.
    • Verify installation: 
      psql --version
  • Set Up VPN Access:
    • Use a service like OpenVPN. Download the client and follow the configuration instructions provided by your VPN administrator.

Step 2: Setting up FastAPI Backend

  1. Clone the repository:
     git clone <backend-repository-url>
  2. Navigate to the backend directory:
     cd backend
  3. Create a virtual environment:
     python -m venv venv
  4. Activate the environment and install dependencies:
     source venv/bin/activate (Linux/macOS)
     venv\Scripts\activate (Windows)
     pip install -r requirements.txt
  5. Run the FastAPI server:
     uvicorn main:app --host 127.0.0.1 --port 8000

Step 3: Setting up Application Layer

1. Clone the Application Layer repository:
 git clone <application-repository-url>
2. Navigate to the application directory:
 cd application-layer
3. Install dependencies:
 npm install
4. Start the application:
 npm start
5. Ensure the application is running at the specified port.

Step 4: Setting up Database Layer

1. Clone the Database Layer repository:
 git clone <database-repository-url>
2. Navigate to the database directory:
 cd database-layer
3. Set up the database:
 psql -U <username> -d <database-name> -f schema.sql
4. Populate initial data (if required):
 psql -U <username> -d <database-name> -f seed.sql
5. Verify the database is running and accessible.

Step 5: Configuring NGINX

  1. Install and start NGINX:
     sudo apt install nginx
     sudo service nginx start
  2. Configure NGINX for reverse proxy:
     Edit /etc/nginx/sites-available/default and add:
     server {
        listen 80;
        server_name localhost;
        location / {
            proxy_pass http://127.0.0.1:8000;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        }
     }
  3. Test and reload configuration:
     sudo nginx -t
     sudo service nginx reload

Step 6: Deploying and Testing

  1. Deploy the backend as a service:
     Create /etc/systemd/system/fastapi.service with:
     [Unit]
     Description=FastAPI Application
     [Service]
     User=your-username
     Group=your-group
     WorkingDirectory=/path/to/backend
     ExecStart=/path/to/venv/bin/uvicorn main:app --host 127.0.0.1 --port 8000
     [Install]
     WantedBy=multi-user.target
  2. Enable and start the service:
     sudo systemctl enable fastapi
  3. Test both web and desktop apps.
